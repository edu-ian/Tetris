Tetris Java – Projeto Acadêmico

Este projeto é uma implementação completa e funcional do jogo Tetris desenvolvida em Java (versão 21) utilizando Swing. O código base foi fornecido pela professora e posteriormente expandido com novas funcionalidades, melhorias estruturais e opções de personalização.

Funcionalidades Principais Mecânicas do Jogo

Movimentação completa das peças: esquerda, direita, queda suave, queda rápida e rotação.

Detecção de colisão e fixação da peça no tabuleiro.

Remoção de linhas completas e reorganização das linhas superiores.

Sistema de pontuação, linhas acumuladas e cálculo automático de nível.

Aceleração gradual conforme mais linhas são removidas.

Timer responsável pela queda das peças.

Personalizações Adicionadas

Seleção de dificuldade: Fácil, Médio, Difícil e Insano.

Modo colorido clássico ou modo monocromático escolhido pelo jogador.

Modo claro e modo escuro do tabuleiro.

Função de pausa.

Sombra da peça (ghost piece), indicando onde ela cairá.

Sistema de "hold" (guardar peça), acionado com a tecla Q.

Interface reorganizada para melhor experiência do usuário.

Como Executar

Instale Java 21.

Abra o projeto no IntelliJ IDEA.

Compile o projeto.

Execute a classe:

tetris.ui.Main

Controles Ação Tecla Mover para a esquerda ← Mover para a direita → Acelerar a peça ↓ Rotacionar ↑ Hard Drop Espaço Pausar P Guardar peça (Hold) Q Créditos da Equipe
